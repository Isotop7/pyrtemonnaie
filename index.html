What is pyrtemonnaie?

Pyrtemonnaie is a wip tool to collect your monthly costs in a simple and intuitive way. Its build on python and uses pyqt to generate a qt-bases ui.

As this is wip and I'm using this to learn python, this project is highly experimental.
